Agent Editor

The app lets users select from a list of agents taken from a database and edit their information. When the edit button is pressed the text fields become editable and when save is clicked the database is updated with information from the fields. Delete will remove that agent from the database. I used IntelliJ IDEA, Scenebuilder, and SSMS
